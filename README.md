# 📊 Retail Sales Performance Dashboard

## 🔹 Project Overview
This project analyzes retail sales data using Excel and Power BI to uncover insights on customer behavior, product performance, and revenue trends.

---
## 🔹 Workflow:
Excel → Data Cleaning → Pivot Analysis → Power BI Dashboard → Insights

---

## 🔹 Tools Used
- Excel (Data Cleaning & Preparation)
- Power BI (Dashboard & Visualization)

---

## 🔹 Key Metrics
- Total Revenue
- Total Quantity Sold
- Average Basket Value
- Total Customers

---

## 🔹 Dashboard Features
- Revenue by Category
- Revenue by Shopping Mall
- Monthly Revenue Trend
- Revenue Distribution by Gender
- Interactive Filters (Category, Gender, Month, Age Group)

---

## 🔹 Key Insights
- Clothing category generates the highest revenue
- Mall of Istanbul is the top-performing location
- Female customers contribute ~60% of total revenue
- Sales show a declining trend after Q1

---

## 🔹 Dataset Source
https://www.kaggle.com/datasets/mehmettahiraslan/customer-shopping-dataset
